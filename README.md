# Student Marks Analysis using Python

This project analyzes real student performance data from the UCI Machine Learning Repository.
The goal is to explore, clean, analyze and visualize student marks using Python without using Excel.

## Dataset Source

UCI ML Repository - Student Performance Dataset (ID: 320)

## Tools Used

-   Python
-   pandas
-   matplotlib
-   seaborn
-   ucimlrepo

## Project Tasks

-   Fetch dataset using UCI API
-   Reconstruct full dataset
-   Clean and validate marks
-   Calculate mean, median, min, max
-   Create bar chart of average marks
-   Create histogram of final exam marks
-   Write 5 data-driven insights

## Output

-   analysis.py
-   insights.txt
-   charts folder containing bar chart and histogram
